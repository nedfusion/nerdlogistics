🚀 UPLOAD INSTRUCTIONS:

1. LOG INTO CPANEL
   - Go to your hosting cPanel
   - Open "File Manager"

2. NAVIGATE TO YOUR WEBSITE FOLDER
   - Usually: /public_html/
   - For subdomain: /public_html/subdomain/

3. CLEAN THE FOLDER
   - Delete all existing files (backup first!)

4. UPLOAD ALL FILES FROM THIS FOLDER
   - Select ALL files in this deployment-files folder
   - Click Upload button in File Manager
   - Wait for upload to complete

5. VERIFY UPLOAD
   - Check that .htaccess is present (show hidden files)
   - Verify assets/ folder with all files
   - Ensure index.html is in root

6. SET UP SSL
   - Go to cPanel → SSL/TLS Status
   - Enable AutoSSL or Let's Encrypt

7. CREATE ADMIN USER
   - See CREATE_SUPER_ADMIN.md in main folder
   - Or check Supabase dashboard

8. TEST YOUR SITE!
   - Visit: https://yourdomain.com
   - Admin: https://yourdomain.com/admin/login
